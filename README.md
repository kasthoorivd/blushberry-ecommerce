# BlushBerry E-Commerce Website

A full-stack e-commerce web application built using the MERN-related backend stack with **Node.js**, **Express.js**, **MongoDB**, and **EJS** templating engine.

## Features

### User Side

* User authentication and authorization
* Product browsing and searching
* Product categories and filtering
* Shopping cart management
* Wishlist functionality
* Secure checkout process
* Order management
* Coupon and offer support
* Razorpay payment integration
* User profile management

### Admin Side

* Admin dashboard
* Product management
* Category management
* Order management
* User management
* Sales reports
* Offer and coupon management

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* Bootstrap / CSS
* JavaScript
* Express Session
* Passport.js / Google Authentication
* Razorpay

## Installation

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Navigate to the project folder

```bash
cd your-project-name
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file and add required environment variables

```env
PORT=3000
MONGODB_URI=your_mongodb_connection
SESSION_SECRET=your_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```

5. Start the server

```bash
npm start
```
